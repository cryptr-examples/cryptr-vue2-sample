# Cryptr with Vue2

## 02 - Add Cryptr SPA JS

Cryptr SPA JS will provide **all the features you need** to your app while respecting authentication standards such as **OAuth2 & Proof Key Exchange**.

To add this library:

```bash
yarn add @cryptr/cryptr-spa-js
```
