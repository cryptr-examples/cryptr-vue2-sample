# Cryptr with Vue2

## 02 - Add Cryptr SPA SDK

Cryptr SPA SDK will provide **all features you need** to your app with a respect of authentication standards like **OAuth2 & Proof Key Exchange**.

To add this library

```
yarn add @cryptr/cryptr-spa-js
```
